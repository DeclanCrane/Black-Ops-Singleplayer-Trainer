# Black Ops Singleplayer Trainer

Feature-rich trainer for Black Ops Singleplayer/Zombies on PC

## Features
* Box ESP
* Teleport Aimbot
* NoClip
* Godmode
* Weapon Spawning / Unlocked Hidden Weapons
* Infinite Money
* Infinite Ammo
* And more!

### How to Compile

Make sure to build in x86, x64 not supported
